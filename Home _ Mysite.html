Light Pole Distribution Map of Triple Cities

Google street view image information extraction
​
Google street view images provide us with a good resource of urban scenes. With the development of computer vision, we can extract useful information from these images automatically and efficiently.
This research uses YOLO to build a deep learning model to extract light poles.  2000 street view images were labeled to train the model, and we got the overall accuracy of 94%.
Finally, I created a light pole distribution map of Triple Cities in Broome County, NY.
Land use classification using Google Street View Images
Urban land use is essential to urban planning and management. However, traditional land use classification methods mainly based on domain experience which is expensive and time confusing.
This project aims to use a convolutional neural network  to classify google street view images to different land use types. 
VGG16 is used to train the classification model. 5000 google street view images divided into 6 classes (Commercial, Others, Parking, Residential, Green, No) were used to train the model. 
The overall accuracy of our model reaches to 82%, and the residential gets the best accuracy of 91% while the parking lot get the lowest accuracy of 57%.
Finally, I created the land use map of Triple cities in Broome County, NY.

Big Data

Quantify walkability with publicly available data
In this study, we proposed a computational framework for walkability measurement. Three major steps of this framework include web scraping of publicly available online data, determining varying weights of variables, and generating a synthetic walkability index.

The pattern of synthetic walkability index of Triple Cities

 

Volunteered geographic information (VGI)

Road Anomaly detection using Mobile phone in-built sensors
In this study, we use GPS and acceleration sensors built in mobile phones to detect anomalies on road. Robust peak detection algorithms and pattern filters are used to detect anomalies. Then I tested different operating systems and different speed, the result shows that the accuracy of the IOS system with an acceptable high speed can reach up to 90% which is much higher than Android system at the same speed.
Finally, an anomaly map of Binghamton university is created using the smartphone with IOS system mounted on a scooter.
This method makes the road condition evaluation more efficient and we can apply it to a larger scale.
